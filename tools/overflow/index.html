Overflow

1. Visible: Valor padrão. O contéudo não é cortado e pode ser redenrizado para fora da caixa de conteúdo.

2. Hidden: O conteúdo é cortado e nenhuma barra de rolagem é exibida.

3. Scrooll: O conteúdo é acessível através de barras de rolagem que são exibidas mesmo que o conteúdo não precise.

4. Auto: Exibe barras de rolagem apenas quando necessário.
